ABOUT SYNERGY

Synergy is a coding and project presentation event organized by IEEE Signal Processing and Computer Society Joint Chapter UP Section and IEEE Signal Processing Chapter IIIT Allahabad. With the intention of developing the skills of analyzing and processing real world signals and thus transforming the tech-geeks from mere comsumer to the producers of the domain knowledge, we are organizing a two phase national level event in coding and signal processing.

The domains which are going to be principle focus of the event are: Image signal processing, Speech signal processing, Computational neuroscience and Physiological signals. The event is divided into two basic phases as follows:

Phases

Phase 1 : Coding Round

Date : 23rd October, 2016

This is learning cum assessment round in which participants will be provided some practice materials cum tutorials as per their domain preference as a home work to learn elementary concepts of signal processing. Based on the tutorial a coding event will be held on 23rd October comprising of two questions for two hours. After this assessment top 10 team will get selected for phase-2.

Phase 2 : Project Presentation and Demonstration Round

Date : 14-15th January, 2017

In this phase some PhD participants will get selected from all over India based on academic competency and they will get associated with the ten selected teams. Every team will be given a real life problem to solve with 75 days in hand. Since it is a collaborative work, so to provide platform for virtual collaboration each team will be provided its personal GitHub repository. On 14th & 15th January, 2017 all 10 teams will mark their presence in IIIT-Allahabad for project presentation and demonstration.
